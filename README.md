# Personal Assistant

A single-user personal AI assistant built on Claude with tool use, MCP connectors,
and a Postgres + pgvector memory store.

Start here: **`CLAUDE.md`** is the project guide (Claude Code reads it automatically).
Full plan: `docs/mvp_plan.md`. Device connectivity: `docs/device_coverage.md`.

## Quick start
1. `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and add your key.
3. `python src/assistant_starter.py`
